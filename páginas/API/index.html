  • Antes de enviar mensagens, é necessário o cadastro do token vinculado à conexão que enviará mensagens.

Seguem abaixo a lista de informações necessárias para envio das mensagens de texto:

Endpoint: http://localhost:8080/api/messages/send
Método: POST
Headers: Authorization (Bearer Token) e Content-Type (application/json)
Body: {“number”: 5599999999999”, “body”: “Sua mensagem”}

Seguem abaixo a lista de informações necessárias para envio das mensagens de texto:

Endpoint: http://localhost:8080/api/messages/send
Método: POST
Headers: Authorization (Bearer Token) e Content-Type (multipart/form-data)
FromData:
  • O número para envio não deve ter máscara ou caracteres especiais e deve ser composto por:
  • Código do país
  • DDD
  • Número
  • number: 5999999999999
  • medias: arquivo
Para realizar o cadastro acesse o menu “Conexões”, clique no botão editar conexão e insira o token nno devido campo.
